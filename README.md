### Installation

**Either download the project or clone the project.**<br/>
**Open VsCode Terminal or PowerShell.**
```
git clone https://github.com/strbbrn/Flutter-Wordpress-Blog.git
cd your-project-dir
flutter pub get
flutter run
```
**Done.**<br/>
